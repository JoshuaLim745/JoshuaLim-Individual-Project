Project Aim
1. Develop a functional Othello board with the appropriate game logic
2. Research, create, and test heuristics that are capable of playing Othello at varying skill levels.
3. Make the heuristics play against each other and record their win, loss and draw rates to each of the other Othello heuristics. 
4. Develop a writeup that will display not only the results of the matches between heuristics but also explain the development process to getting to that stage.


Heuristics Chosen
1. Random Move
2. Naive Greedy - Most Tiles Flipped
3. Weighted Tile Positions
4. Frontier Disk

Libraries used

3. copy
4. random
5. time
